# Python >= 3.9
# 
# ============================================================================
# INSTALACIÓN EN DIFERENTES SISTEMAS OPERATIVOS
# ============================================================================
# 
# WINDOWS:
# 1. Instalar Python 3.8+ desde python.org
# 2. pip install -r requirements.txt
# 3. (Opcional) pip install pywin32 para funcionalidades de Windows
# 
# UBUNTU/DEBIAN:
# 1. sudo apt update
# 2. sudo apt install python3 python3-pip python3-dev
# 3. sudo apt install python3-pyqt5 python3-pyqt5.qtsvg  # Opcional, desde repos
# 4. pip3 install -r requirements.txt
# 
# MACOS:
# 1. Instalar Python 3.8+ desde python.org o usar Homebrew
# 2. pip install -r requirements.txt
# 
# ============================================================================
# SOLUCIÓN DE PROBLEMAS COMUNES
# ============================================================================
# 
# Error con PyQt5:
# - Windows: Instalar Microsoft Visual C++ Redistributable
# - Linux: sudo apt install python3-pyqt5-dev
# - macOS: brew install pyqt5
# 
# Error con Pillow:
# - Instalar dependencias de imagen del sistema
# - Ubuntu: sudo apt install libjpeg-dev zlib1g-dev
# 
# Error con cryptography:
# - Requiere Rust toolchain en algunos sistemas
# - Ubuntu: sudo apt install build-essential libssl-dev libffi-dev
# 
# ============================================================================
